# Ascend — Backend Foundation

```
server/
├── config/db.js
├── controllers/   (empty, next phase)
├── routes/        (empty, next phase)
├── models/        (empty, next phase)
├── middleware/errorHandler.js
├── server.js
├── .env.example
└── package.json
```

## Setup
```
cd server
npm install
cp .env.example .env   # fill MONGO_URI
npm run dev
```
